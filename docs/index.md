# Koimux Bot

在手机上部署 Koishi 的完整工具集

::: warning Termux 兼容性说明
Termux 由于架构问题，不可避免会在环境上存在部分兼容性问题。
:::

## 快速开始

推荐 **初次使用的用户** 直接使用 [Koishi Manager (koimuxTUI)](./scripts/koimuxTUI) 进行安装。

## 核心特性

| 特性         | 说明                                                  |
| ------------ | ----------------------------------------------------- |
| 📱 移动端部署 | 在 Android 手机上轻松部署 Koishi，无需电脑            |
| 🚀 一键安装   | 使用 koimuxTUI 脚本，自动化安装所有依赖和 Koishi 实例 |
| 🛠️ 完整工具链 | 包含 Chromium、FFmpeg 等完整依赖，开箱即用            |
| 📦 实例管理   | 支持多实例管理、备份恢复，轻松管理你的 Koishi         |
| 🔧 容器支持   | 支持 proot 容器，突破 Termux 限制                     |
| 📚 详细文档   | 提供完整的安装和使用教程，新手友好                    |
